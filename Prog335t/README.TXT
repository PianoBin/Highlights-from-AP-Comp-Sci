Program 335t
(prime sandwich)

Program Description:  Write a program that inputs an integer, "N",  greater  than 2 and then finds the largest prime that is less than or equal to "N" and the smallest prime that is greater than or equal to "N".  To simplify the process, include a method "isprime" to help you solve the problem.
After completing the first calculation query the user to determine if she would like to check another number.
